库

动画

  • Bounce.js，Bounce js动画，很Q弹
  • Dynamics.js，带有物理效果的动画
  • Choreographer.js，动画库，能够处理复杂的css
  • anime.js，动画库
  • skrollr，滚动动画
  • Granim.js，渐变动画，超帅
  • voxelcss，不知道怎么形容，3d就对了
  • ramjet，dom转换动画
  • superplaceholder.js，placeholder动画
  • vanilla-tilt.js，3d视差移动
  • Snap.svg，svg动画库

ui相关

  • Slideout.js，滑动菜单的效果，用于手机
  • Zooming，图像放大效果
  • NProgress.js，头部加载条
  • Slick.js，轮播
  • lightgallery.js，图片幻灯片
  • Swiper，滑动幻灯片

滚动相关

  • Scrollreveal.js，滚动效果
  • moveTo.js，移动到页面相应位置(回到顶部)
  • ScrollTrigger，滚动触发

弹出

  • Popper.js，轻量的弹出框，不同于modal的弹出
  • sweetalert,js，弹出提示
  • Notyf，轻量提示框

工具

  • Dropzone.js，拖拽上传
  • UAParser.js，分析浏览器UA
  • uaop，ecomfe写的aop库
  • meld.js，aop库，比较好用
  • appmetrics.js，分析页面信息然后上传到google analysis
  • List.js，处理列表的库，支持搜索排序等功能
  • Cleave.js，表单输入的时候自动格式化
  • qart.js，二维码，支持自定义
  • Just，工具函数库，是一堆独立的库，功能类似lodash
  • clipboard.js，最好用的复制粘贴，支持ie9以上
  • hammer.js，移动端触摸库
  • badjs，前端性能监测
  • re-build，用自然语言写正则表达式
  • flatpickr，最好用的日历选择器

for fun

  • impress.js，类似ppt，设计感爆炸的ppt
  • Rythm.js，声音控制节奏移动dom
  • sphinx，将字符串转化为base64编码并生成图像
  • baffle.js，很炫的文字模糊显示效果
