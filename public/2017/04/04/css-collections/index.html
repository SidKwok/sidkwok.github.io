参考标准

  • Can I Use，浏览器兼容性查询，也可以查javascript
  • CSS Reference，css的参考手册
  • HTML Reference, html的参考手册

库

  • Animate.css，各种css动画的集合
  • Hover.css，Hover动画的集合，一般用于button的hover效果
  • Effeckt.css，一些css动画的集合，不是一个库，只是一些例子
  • Hint.css，hover提示，如果商用要给钱
  • CSS ARROW，用css写箭头，一个生成工具
  • CSSPIN，用css写的各种loader，十分有趣！
  • Wenk，hover提示
  • hamburgers，hamburger icon，还有动画效果
  • StarWarsIntro，星球大战开场字幕效果
  • SPINNERS，css加载动画，不过这不是一个库，是一堆css集合
  • CSS Loader，css加载动画，这个是占满整个屏幕的，而不是单单一个icon
  • parallax，移动鼠标屏幕会跟着3d移动
  • bttn.css，button样式集合
  • CSShake，疯狂摇摆
  • balloon.css，提示框
  • loaders.css，一堆css加载动画

Icon

  • 阿里巴巴矢量图标库，可以搜索相关的icon并自定义

  • weather-icons，天气图标
  • bytesize-icons，svg icons
  • CSS ICON，可以直接复制一段css生成icon

工具

  • WebGradients，css做出来的渐变色彩
  • css3 animation，css3的动画制作
  • sass 中文文档
  • clippy，使用clip来裁剪图像的css代码
  • less转css
  • Emoji Searcher，emoji搜索器，主要用于git的commit
  • favicon.io，favicon制作器
  • grid-kiss，post-css的栅栏系统的插件
  • Android Material 材料风格图标LOGO生成器
